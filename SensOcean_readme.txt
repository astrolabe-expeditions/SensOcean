Utilisation du boitier SensOcean : 

Par défault lorsque le boitier s'allume, il est en mode acquisition de données, il enregistre continuellement les données sur la carte SD. 

Configuration du boitier : 
----------------------------
Pour modifier la configuration du boitier il suffit de modifier le fichier config.txt sur la carte SD
id_logger : numéro marquer sur le boitier à l'intérieur. 
delay_series : durée en secondes de mise en veille entre chaque series de mesures
number_measures : nombre de mesure durant chaque serie

Par d'espace avant ou aprés le =, et retour à la ligne directement aprés la valeur (sans espace à la fin).
